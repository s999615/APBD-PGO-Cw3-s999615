# System zamowien kawiarni

Prosta aplikacja konsolowa napisana w Javie, przygotowana w ramach zajec z PGO.  
Program symuluje niewielki system obslugi zamowien w kawiarni.

## Funkcjonalnosci

- tworzenie produktow dostepnych w menu,
- tworzenie klientow,
- skladanie zamowien,
- dodawanie produktow do zamowienia,
- obliczanie wartosci zamowienia,
- liczenie produktow w zamowieniu,
- oznaczanie zamowienia jako oplaconego,
- porownywanie obiektow za pomoca metody `equals()`.

## W projekcie zostaly wykorzystane:
- klasy i obiekty,
- konstruktory,
- ArrayList,
- pola i metody statyczne,
- nadpisanie metod `toString()` oraz `equals()`.

## Struktura projektu

Projekt sklada sie z nastepujacych klas:
- `ProduktMenu`
- `KlientKawiarni`
- `Zamowienie`
- `Main`

## Przykladowe dzialanie programu

Program tworzy przykladowe produkty, klienta oraz zamowienie, a nastepnie wyswietla podsumowanie w konsoli wraz z laczna wartoscia zamowienia i statusem platnosci.
